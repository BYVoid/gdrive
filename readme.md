Google Drive Filesystem
=======================

This project contains two parts: libgdrive and gdrivefs.

* libgdrive is a C++ wrapper of Google Drive/Docs API.
* gdrivefs uses libgdrive and fuse to imeplement filesystem interfaces.

Google Drive Filesystem

Requires
========

libgdrive

* libcurl
* libxml2

gdrivefs

* libgdrive
* fuse (fuse4x for Mac)
